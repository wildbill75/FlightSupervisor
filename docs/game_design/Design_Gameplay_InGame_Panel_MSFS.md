# Design Gameplay: In-Game Toolbar Panel

## Overview
The MSFS Toolbar Panel allows pilots to monitor their Flight Supervisor status directly within the cockpit, which is particularly essential for VR users and single-monitor setups.

## 📱 Features
- **SuperScore HUD**: Real-time display of the current score and flight phase.
- **Ground Ops Tracker**: Visual progress bars and status messages for the ongoing turnaround.
- **Alert Notifications**: Critical safety or comfort warnings (e.g., "Overspeed", "Severe G-Force") are mirrored from the main application.

## 🕶️ VR Optimization
- **High Contrast**: Neon blue and green accents on a dark charcoal background.
- **Minimalism**: Focuses only on actionable data to avoid cluttering the pilot's field of view.
- **Interactive Buttons**: (Future) Capability to trigger PA announcements or interact with Ramp Agents directly from the toolbar.
